Auth

  • /user/login

    Post method

    Must have on the body this following parameters: password, username
    Example:

    {
    "password":"user", required
    "username":"user" required
    }

  • /user/logout

    Delete method

    No body required

Task

  • /task/all

    Get method

    No url parameter need or body

  • /task/unique/:taskId

    Get method

    url parameter need making reference to the taskId which wants to consult

  • /task/delete/:taskId

    Delete method

    url parameter need making reference to the taskId which wants to eliminate

  • /task/new-task/

    Post method

    Body needed to create the Task
    Example:

    {
    "title":"", required
    "description":"", required
    "status":"", required
    "tags":"",optional
    "manager":"", optional
    "comment":"" optional
    }

  • /task/modified-task/:taskId

    Put method

    Body needed to change values of the Task

    Url parameter taskId needed to identified the task to update

    If not pass any applicable value will return a message indicating the applicable values
    Example of available attributes

    {
    "title":"", optional
    "description":"", optional
    "status":"", optional
    "tags":"",optional
    "manager":"", optional
    "comment":"" optional
    }
