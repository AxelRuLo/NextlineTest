# Task manager
## instructions

1 - npm install

2 - node server.js

.env file its necessary to connect whit the database, must have the following attributes and must go on the root:

USERDATABASE=''

PASSWORD='%'

DB=''

HOSTDB=''

PORTDB=''

SECRET=''
